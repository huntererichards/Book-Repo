# Book-Repo
Note: This project was developed using Visual Studio 2022.

This project is a  C++ application that manages a small repository of books. The program demonstrates object-oriented programming concepts such as classes, constructors, destructors, enumerations, dynamic memory allocation, parsing, and repository management.

The application stores books in a repository, displays book information, filters books by type, validates book IDs, calculates average prices, and removes books from the repository.

The program does the following:

*Parse raw book data strings into structured objects

*Store books dynamically in a repository

*Display all books

*Filter books by defined categories

*Detect invalid book IDs

*Calculate average book prices

*Remove books by ID

*Demonstrates constructors/destructors and dynamic memory cleanup
